# Бие даалт 2: Улаанбаатар хотын замын уулзварын түгжрэлийг загварчлах

## Зорилго

Стохастик системийг Монте-Карло аргаар загварчилж, замын хөдөлгөөний ачаалал, хүлээлтийн хугацаа болон гэрлэн дохионы оновчтой мөчлөгийг тооцоолох.

## Файлын бүтэц

- `src/model.py`: Загварын үндсэн тэгшитгэлүүд, ирэлт үүсгэх, 1 уулзварын симуляц.
- `src/simulation.py`: Монте-Карло давталтууд, бүтэн өдрийн болон 4 чиглэлийн симуляц.
- `src/analysis.py`: Статистик тооцоолол, итгэлцлийн завсар, оновчлол.
- `src/visualization.py`: Үр дүнг зураглах (matplotlib).
- `tests/`: Unit test-үүд.

## Ажиллуулах заавар

1. Сангуудыг суулгах: `pip install -r requirements.txt`
2. Тестүүдийг шалгах: `pytest tests/ -v`
3. Үндсэн симуляцыг ажиллуулах: `python src/simulation.py` (эсвэл тусдаа script бичиж ажиллуулна)
